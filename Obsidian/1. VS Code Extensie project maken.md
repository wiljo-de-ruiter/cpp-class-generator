# Stap 4: Test de extensie

1. Druk op ==F5== in VS Code om een nieuwe "Extension Development Host" te starten.
2. Open een ==.cpp== bestand
3. Druk op ==Ctrl+Shift+P==, typ ==Create C++ Class== -> voer een naam in -> class wordt gemaakt

# Doel

```

```
